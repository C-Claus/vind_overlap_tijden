# vind_overlap_tijden
Een klein programma waarin medewerkers hun afwezigheden kunnen opgeven. Het programma moet vervolgens kunnen tonen of afwezigheidsperiodes elkaar overlappen.

# gebruikte tools
Python 3.8
Django 3.0.6
